# **Simple calculator** for terminal

This calculator has many of functions,it's has **3 menu:**
- Arithmetic menu:
  - addition
  - subtraction
  - multiplication
  - division
- Trigonometric menu:
  - Sinus of Degree
  - Cosinus of Degree
  - Tangens of Degree
  - Convertation of degree to radian
  - Convertation radian to degree
- Mathematic menu:
  - exponentiation
  - Square Root
  - Logorithm
  - Comparison(max,min)
